---
description: Deploy to Vercel
---

1. Ensure you are logged in to Vercel CLI:
   ```bash
   npx vercel login
   ```

2. Deploy the application:
   ```bash
   npx vercel
   ```
   - Follow the prompts.
   - Set up and deploy? [Y]
   - Which scope? [Select your account]
   - Link to existing project? [N]
   - Project name? [swp-club]
   - In which directory? [./]
   - Want to modify these settings? [N] (We already configured them)

3. Your app will be deployed!
